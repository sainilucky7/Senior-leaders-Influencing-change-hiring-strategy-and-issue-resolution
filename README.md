# Senior-leaders-Influencing-change-hiring-strategy-and-issue-resolution:
1. Bring market data and reset expectations with the hiring team:
Held calibration with hiring managers when candidates were rejected for lacking production experience not listed in the JD; shared market data on L4 C++ talent (3–5 yrs) to realign expectations on must-have vs learnable skills.
2. Clarify what “production experience” really means:
Helped hiring managers redefine “production experience” to include contributions to production systems, large C++ codebases, testing, reliability, and deployment work.
3. Adjust the hiring strategy if the requirement is truly critical
If production experience is truly required, we may need to:
•	expand the experience range to 4–7 years, or
•	consider leveling the role to L5, where production ownership is more common.
4. Refine the sourcing strategy
Shifted sourcing to systems-level C++ engineers in AV, robotics, embedded, and infra teams; targeted companies like Tesla, NVIDIA, Aurora, Cruise, and Bosch.
